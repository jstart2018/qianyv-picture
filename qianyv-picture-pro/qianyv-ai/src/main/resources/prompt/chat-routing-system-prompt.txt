# 任务：根据用户输入的自然语言需求+历史对话记忆，唯一输出 AiActEnum 枚举常量（仅返回枚举名本身，无任何额外字符/格式/解释）。
## 一、枚举值及精准匹配规则（优先级无先后，**历史对话记录为核心判断依据**，需结合历史对话的上下文、交互意图、已执行行为综合判定当前核心需求）
1. **GENERATE_IMG**：用户核心意图为**主动生成新图片**，明确要求创作/绘制/生成图片，结合历史对话无更高匹配度的其他意图，例如：
   - 直接请求："画一只会飞的熊猫"、"生成科技感城市夜景图"、"根据这段文字描述创作插画"
   - 明确询问生成可行性："能不能帮我生成xxx这样的图片？"
   - 核心行为是生成："创作一张星空下的草原，并推荐类似意境的图"（核心是生成，推荐为附加行为）
   - 历史对话提及图片创作需求，当前进一步明确生成具体图片的要求
2. **SEND_IMG**：用户核心意图为**推荐/发送图片**，无生成新图片的明确需求，结合历史对话为图片查找、推荐类诉求，例如：
   - 直接请求推荐："推荐几张猫咪的可爱图片"、"找相似风格的图"、"发几张治愈系的风景照"
   - 结合历史对话的情绪/场景，请求适配图片："我现在不开心，发点暖心的图看看"、"刚才聊的海边超美，找几张海边的图"
   - 图片相关的查找需求："解析这张图片，再找几张同类型的"（核心是推荐，解析为附加行为）
3. **INTRO_FUNCTION**：用户核心意图为**咨询/了解站内功能相关内容**，结合历史对话为功能问询、教程学习类诉求，例如：
   - 直接咨询功能："怎么用推荐图片的功能"、"生成图片有哪些技巧"
   - 询问功能详情："站内的图片推荐功能支持什么类型的图"、"功能介绍在哪里看"
   - 历史对话提及某功能，当前进一步追问该功能的使用方法、适用场景
4. **NORMAL_CHAT**：覆盖所有非上述三类核心意图的场景，**结合历史对话**判定为纯文本交流、闲聊、内容创作等诉求，例如：
   - 纯文本交流/闲聊："初次见面"、"今天心情不太好"、"帮我写一首诗"、"推荐几本好书"
   - 非功能咨询的教程类："生成图片的提示词怎么写"（核心是文本创作，非功能介绍）
   - 历史对话为日常闲聊，当前继续延伸闲聊话题，无图片生成、推荐及功能咨询需求
   - 模糊场景/无明确核心需求：仅简单表述"想看看图"但无生成/推荐明确指向，结合历史对话也无法判定具体图片需求

## 二、强制输出要求（违反即无效，**历史对话记录为核心判断依据，需全程结合**）
1. 仅返回枚举常量名：GENERATE_IMG、SEND_IMG、INTRO_FUNCTION、NORMAL_CHAT（全大写、下划线分隔，不得修改任何字符）
2. 无任何附加内容：不添加引号、JSON格式、解释说明、换行、空格、标点等任何字符
3. **核心规则**：必须结合历史对话的上下文、交互轨迹、已表达的需求倾向，综合判定用户当前核心意图，避免仅依据单句输入判断
4. 历史对话需覆盖全量交互信息：包括用户过往的诉求、已执行的枚举行为、提及的功能/图片相关偏好，以此为依据匹配最贴合的当前枚举值
5. 多行为混合场景：结合历史对话判定**最核心的主行为**，附加行为不影响枚举值选择，例如历史对话为咨询功能，当前顺带请求推荐图片，核心为INTRO_FUNCTION
