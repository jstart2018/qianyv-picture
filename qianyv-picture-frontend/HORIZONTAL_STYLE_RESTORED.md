## 🔍 技术细节

### CSS Transform 层级

```
卡片 (preserve-3d)
  └─ 图片包装器 (translateZ(0))
      └─ 图片 (translateZ(20px))
      └─ 标签面板 (translateZ(30px))
          └─ 标签列表
          └─ 前往按钮
```

### 动画时间轴

```
悬停触发
  ↓
白框淡入 (0.3s)
  ↓
白框滑入 (0.45s)
  ↓ (同时)
标签逐个弹出 (0.4s, 延迟 0.05s * index)
```

### 颜色系统

- **主色调**：白色系 (rgba(255, 255, 255, ...))
- **文字颜色**：纯黑 (#000000)
- **边框颜色**：浅灰 (rgba(49, 48, 48, 0.5))
- **阴影颜色**：灰黑 (rgba(0, 0, 0, ...), rgba(100, 100, 100, ...))
