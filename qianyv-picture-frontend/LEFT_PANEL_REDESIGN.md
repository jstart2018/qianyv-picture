# 左侧用户信息区域重新设计

## 🎯 设计需求

根据用户要求，完全重新设计了左侧用户信息区域，实现了**矩形区域 + 内部卡片**的布局结构。

---

## 📐 布局结构

### 整体布局（三栏设计）

```
┌──────────────────────────────────────────────────────────────────┐
│  顶部导航栏 (60px) - 不受影响                                       │
├────────────────────┬──────────────────────┬──────────────────────┤
│                    │                      │                      │
│   左侧矩形区域      │    中部博客列表       │   右侧矩形区域        │
│   (透明背景)       │    (可滚动)          │   (透明背景)         │
│   宽度: 33.33%     │    宽度: 41.67%      │   宽度: 25%          │
│                    │                      │                      │
│   ┌────────────┐   │                      │                      │
│   │            │   │  博客内容 1          │   排行榜卡片          │
│   │  用户卡片   │   │                      │   (待优化)           │
│   │ (虚线框)   │   │  ━━━━━━━━━          │                      │
│   │            │   │                      │                      │
│   └────────────┘   │  博客内容 2          │                      │
│                    │                      │                      │
│   固定不滚动        │  (独立滚动)          │   固定不滚动          │
│                    │                      │                      │
└────────────────────┴──────────────────────┴──────────────────────┘
  ↑                                            ↑
  透明可见动态渐变                               透明可见动态渐变
```

## ✨ 实现细节

### 1. **左侧矩形区域** (`.left-panel`)

```css
.left-panel {
  flex: 0 0 33.33%; /* 占整个网页的 1/3 */
  height: 100%;
  overflow: hidden; /* 完全不滚动 */
  background: transparent; /* 透明背景 - 显示 App.vue 动态渐变 */
  display: flex;
  align-items: center; /* 卡片垂直居中 */
  z-index: 15;
}
```

**特点：**

- ✅ 宽度占网页的 **1/3**
- ✅ **透明背景**，可见 App.vue 的动态颜色渐变
- ✅ **固定不滚动**
- ✅ 作为容器，内部放置用户卡片

---

### 2. **用户信息卡片** (`.user-info-card`)

```css
.user-info-card {
  /* 尺寸和位置 */
  width: 66.7%; /* 卡片宽度占矩形的 2/3 */
  height: 50%; /* 高度占矩形的 1/2 */
  margin-left: 10%; /* 左边距 10% */
  margin-right: 23.3%; /* 右边距 23.3% */

  /* 外观样式 */
  background: rgba(255, 255, 255, 0.75); /* 白色半透明 */
  backdrop-filter: blur(20px);
  border: 2px dashed rgba(138, 180, 248, 0.4); /* 虚线框 */
  border-radius: 20px; /* 适当圆角 */

  /* 布局 */
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  padding: 32px 24px;

  /* 效果 */
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.08);
  transition: all 0.4s cubic-bezier(0.4, 0, 0.2, 1);
}
```

**特点：**

- ✅ **水平布局**：左边距 10% + 卡片 66.7% + 右边距 23.3% = 100%
- ✅ **垂直居中**：通过父容器 `align-items: center` 实现
- ✅ **虚线框**：`2px dashed` 样式
- ✅ **白色半透明**：`rgba(255, 255, 255, 0.75)`
- ✅ **毛玻璃效果**：`backdrop-filter: blur(20px)`
- ✅ **20px 圆角**

### 4. **交互效果**

```css
.user-info-card:hover {
  border-color: rgba(138, 180, 248, 0.6); /* 虚线框变亮 */
  box-shadow: 0 12px 48px rgba(138, 180, 248, 0.15); /* 阴影增强 */
  transform: translateY(-4px); /* 向上浮动 */
}
```

---

### 5. **右侧区域调整**

```css
.right-panel {
  flex: 0 0 25%; /* 宽度减小到 25% */
  height: 100%;
  overflow: hidden; /* 固定不滚动 */
  background: transparent; /* 透明背景 */
}

.ranking-card,
.chat-card {
  width: 90%; /* 占右侧矩形的 90% */
  margin: 0 auto; /* 水平居中 */
  max-height: calc(100vh - 108px);
  overflow-y: auto; /* 内容可滚动 */
}
```

**特点：**

- ✅ 右侧矩形宽度从固定 320px 改为 **25%**
- ✅ **固定不滚动**（容器）
- ✅ 排行榜卡片内容可独立滚动
- ✅ 与左侧保持一致的设计理念

## 🎨 视觉效果特点

### 1. **透明背景的妙处**

- 左右矩形区域透明
- 可完整展示 App.vue 中设置的**动态颜色渐变**
- 营造出**沉浸式**的视觉体验

### 2. **虚线框设计**

```css
border: 2px dashed rgba(138, 180, 248, 0.4);
```

- 轻盈、透气的视觉感受
- 与半透明白色背景形成对比
- Hover 时虚线变为 `rgba(138, 180, 248, 0.6)`

### 3. **层次结构**

```
第一层：App.vue 动态渐变背景（全屏）
第二层：透明矩形区域（左侧 + 右侧）
第三层：半透明卡片（用户信息 + 排行榜）
第四层：卡片内容（文字、图标等）
```

## 🔍 关键技术点

### 1. **Flexbox 布局**

```css
/* 父容器 */
.home-view {
  display: flex;
  height: calc(100vh - 60px);
}

/* 左侧：固定 1/3 */
.left-panel {
  flex: 0 0 33.33%;
}

/* 中部：自适应 */
.center-panel {
  flex: 1;
}

/* 右侧：固定 1/4 */
.right-panel {
  flex: 0 0 25%;
}
```

### 2. **垂直居中**

```css
.left-panel {
  display: flex;
  align-items: center; /* 让子元素(卡片)垂直居中 */
}
```

### 3. **透明背景**

```css
.left-panel,
.right-panel {
  background: transparent; /* 显示 App.vue 的动态渐变 */
}
```

### 4. **毛玻璃效果**

```css
.user-info-card {
  backdrop-filter: blur(20px);
  -webkit-backdrop-filter: blur(20px);
}
```
