## 🎨 颜色和样式说明

### 左侧矩形区域

```css
背景: transparent (透明，显示 App.vue 动态渐变)
宽度: 33.33% (网页的 1/3)
高度: calc(100vh - 60px)
```

### 用户信息卡片

```css
背景: rgba(255, 255, 255, 0.75) (白色75%透明度)
边框: 2px dashed rgba(138, 180, 248, 0.4) (蓝色虚线)
圆角: 20px
阴影: 0 8px 32px rgba(0, 0, 0, 0.08)
毛玻璃: backdrop-filter: blur(20px)
```

### Hover 效果

```css
边框: rgba(138, 180, 248, 0.6) (虚线变亮)
阴影: 0 12px 48px rgba(138, 180, 248, 0.15) (增强)
位移: translateY(-4px) (向上浮动)
```

## ✨ 视觉层次

```
第1层 (最底层)
    └─ App.vue 动态渐变背景 (全屏铺满)
           ↓
第2层
    ├─ 左侧透明矩形 (Z-index: 15)
    ├─ 中部博客区域 (Z-index: 1)
    └─ 右侧透明矩形 (Z-index: 10)
           ↓
第3层
    ├─ 用户信息卡片 (虚线框 + 半透明白色)
    └─ 排行榜卡片 (半透明白色)
           ↓
第4层 (最上层)
    └─ 卡片内容 (文字、图标、按钮等)
```
